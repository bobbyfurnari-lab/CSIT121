GIF Format Example

About This Image

This animated GIF shows a basketball repeatedly bouncing. It’s a simple but effective loop that captures the motion and rhythm familiar to anyone who’s played the game.

Why GIF Format?

GIF is perfect for short, looping animations like this one. Its limited color range keeps the file size small, while still allowing smooth playback across all modern browsers without the need for video.
